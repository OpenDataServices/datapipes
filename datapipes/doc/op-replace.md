## replace

Find and replace.

### Usage

    replace [-r] [-c columns] [FIND] [REPLACE]

    -r, --regex
        Find argument is a regular expression.

    -c COLUMNS, --columns COLUMNS
        comma-separated list of columns to search.

    FIND
        Text to search for.

    REPLACE
        Text to replace the found text with. Defaults
        to the empty string.
